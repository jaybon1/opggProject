## 라이브러리

1. 구글 머터리얼
2. 서클 이미지뷰
3. 롬복
4. 글라이드

7. LIFECYCLE

1. 게이머 닉네임을 이용하여 전적 검색
2. 상위 유저들의 랭킹 검색
3. 게임 상세 내용 확인
4. 커뮤니티 회원가입
5. 구글 로그인
6. 카카오 로그인
7. 일반 로그인
8. 글쓰기 / 게시글 검색 / 글삭제 / 글수정 / 댓글쓰기 / 댓글삭제
